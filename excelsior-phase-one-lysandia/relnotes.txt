Excelsior Phase One: Lysandia
Version 2.0 Release Notes  4/19/94
------------------------------------

The following is a list of changes made to Excelsior since version 1.0:

 * The game is now fully animated!
 * The font has been redrawn to be more easily read.
 * Mouse support has been reduced to movement only.
 * SoundBlaster support has been enhanced.  You can now specify the address
   and IRQ of the SoundBlaster by running the game with the command:
                        EX -SB220,5
   This would set the game to use the address 220h and IRQ 5.  You must
   supply both parameters when using this option.
 * Light in dungeons and keeps has been modified to a new style.
 * Monster AI has been improved.
 * Saved games can now be more easily transported from one system to another.
 * Various bugs in Excelsior v1.0 have been corrected.

Excelsior v2.0 has been thoroughly tested and should perform just as well on
your system as v1.0 did.

IF YOU ARE USING VERSION 1.0:
If you have registered v1.0, you must use the supplied UPGRADE utility to
maintain your registration.  In addition, you may port your saved games from
version 1.0 to version 2.0 by using the UPGRADE utility.  Read the file
UPGRADE.TXT on how to do this.  Do NOT erase v1.0 from your hard drive prior
to using the UPGRADE utility!!!

If you registered your copy of Excelsior v1.0 and accidentally erased your
registered copy, run the REGISTER program under v2.0 and send us your new
registration code.  We will register your copy of version 2.0 at no charge.
